BuildFire Plugins SDK

Here you will find the help you need to start building your plugin.

Getting Started

  • 1. To get started, sign up for the developer program at http://developer.buildfire.com.
  • 2. Once you have done so, you should be issued an App ID that can be used test your plugin.
  • 3. Open scripts/appContext.js and change the appId in window.appContext.currentApp.
  • 4. Now you can begin developing your plugin in plugins/myPlugin.
  • If you choose to change the folder name, make sure to change it in the scripts/appContext.js file as well.
  • Make sure you run your local environment as a web application environment and not as static files from the file system.
